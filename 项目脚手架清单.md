# 后端脚手架文件清单（Sprint 1）

## 一、Maven 父工程

```
offer-pilot/pom.xml                              ← 父 POM，锁定所有版本
```

## 二、offer-common（公共模块）

```
offer-common/pom.xml
offer-common/src/main/java/com/offerpilot/common/
├── result/
│   ├── Result.java                   ← 统一响应包装
│   ├── ResultCode.java               ← 状态码枚举
│   └── PageResult.java               ← 分页响应

│   ├── GlobalExceptionHandler.java   ← 全局异常处理
│   ├── BusinessException.java        ← 业务异常

├── constant/
│   └── AuthConstant.java             ← JWT、Header 常量
```

# 脚手架搭建顺序（建议）

```
Day 1                    Day 2                    Day 3
├─ 父 pom.xml            ├─ offer-auth CRUD       ├─ offer-user 业务接口
├─ offer-common         ├─ JWT + Security         ├─ offer-application 模块
├─ docker-compose.yml    ├─ Gateway 路由           ├─ 数据库 SQL
├─ 启动 Nacos/MySQL      ├─ 启动验证 auth 链路      ├─ 前端项目初始化
└─ 验证公共模块编译       └─                        └─ 前后端联调验证
```
